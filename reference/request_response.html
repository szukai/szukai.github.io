# CopyUpdateRequest

# Feature Properties

If you register a connector method with the feature properties which match following rules, then you will receive a request which is a dict with data mentioned in Attribute section.

Matching wildcard patterns:

  • storage.*.copy
  • storage.*.update

At least one of parent_id or name will be present.

# Attributes

Name Type Description Note
id str ID of file, folder, etc.
type str file, folder, etc.
name str (Optional) New file/folder name.
parent_id str (Optional) New parent folder ID.
user_id str (Optional) The ID of the user whose folder the file is being moved to. Only admin account support this
